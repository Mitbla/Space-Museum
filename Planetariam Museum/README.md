# Planetariam Museum Virtual Tour

This project is a web-based virtual museum inspired by a planetarium. Users can move around a 3D environment using WASD keys and mouse, exploring the dome and interactive info points.

## Features
- 3D planetarium dome and basic museum layout
- First-person movement controls
- Interactive info points

## How to Run
1. Open `index.html` in your browser.
2. Click anywhere to activate movement controls.
3. Use WASD keys to move and mouse to look around.

## Customization
- Replace placeholder assets with your own models and textures.
- Add more info points and interactive objects in `main.js`.

## Dependencies
- [Three.js](https://threejs.org/) (loaded via CDN)
